# Weather App

## Name: Ibrahim Olabode

## Student ID:  ALT/SOE/BAR/026/0231

## Project Description:   This Weather App was built using HTML, CSS and JavaScript. The application allows users to search for any city in the world and view realtime weather information including temperature, weather condition, humidity, wind speed, UV index, and  a 5-days forecast. This app uses the Open-Metro Geocoding API to convert city name into coordinates and the Open-Meteo Forcast API to retrive weather data. The interface is responsive and works on both desktop and mobile devices. 

Features
Search weather by city name
Current temperature display
Humidity information
Wind speed information
Weather condition and icon display
UV index display
5-Day weather forecast

Technology
HTML5
CSS3
JavaScript (ES6)
Open-Meteo Geocoding API
Open-Meteo Forecast API
